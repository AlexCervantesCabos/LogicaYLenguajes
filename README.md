# Lógica y Lenguajes — modo examen

Herramienta web estática para estudiar preguntas de Lógica y Lenguajes.

## Cómo usarla

Abre `index.html` o publica el repositorio con GitHub Pages.

La herramienta permite:

- Filtrar preguntas por tema.
- Mezclar preguntas.
- Escribir tu respuesta antes de mirar la solución.
- Marcar cada pregunta como sabida, no sabida o duda.
- Guardar el progreso automáticamente en el navegador.

## Publicar con GitHub Pages

1. Entra en el repositorio en GitHub.
2. Ve a `Settings` → `Pages`.
3. En `Build and deployment`, selecciona `Deploy from a branch`.
4. Elige la rama `main` y la carpeta `/root`.
5. Guarda.

Cuando GitHub Pages termine de desplegar, la web estará en:

`https://alexcervantescabos.github.io/LogicaYLenguajes/`

## Atajos

- `Enter`: mostrar solución.
- `1`: la sabía.
- `2`: no la sabía.
- `3`: duda / repasar.
- `→`: siguiente pregunta.
